01The Problem

Printed links should not expire before the material does.

A standard QR code locks the final destination directly into the pattern. Once that code is printed, the link is fixed.

That works until the website changes, the campaign ends, the menu moves, the portfolio is updated, or the event page needs to become a thank-you page.

The printed item is still useful. The link behind it is not.

// status destination locked — reprint required
02How It Works

Relinq separates the printed code from the live destination.

  1. 01Step

    Print one permanent Relinq QR

    Create a QR code that points to a stable Relinq address. The printed pattern stays the same forever.

  2. 02Step

    Control the destination behind it

    Set the real link from your account — website, menu, portfolio, campaign page, contact card, or landing page.

  3. 03Step

    Change it whenever needed

    Update the destination anytime. Every scan of the existing printed QR will route to the latest active link.
